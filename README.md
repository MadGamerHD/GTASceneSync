## GTASceneSync IPL-IDE Blender Addon

### Overview
The **GTASceneSync IPL-IDE** addon for Blender is designed to facilitate the export of 3D models and scenes from Blender into formats used by various Grand Theft Auto (GTA) games. This addon provides functionality to export objects in both the **IPL** (Item Placement List) and **IDE** (IDE Object Definition) formats, which are essential for modding and scene management in GTA titles.

### Key Features
- **Export to IPL (GTA San Andreas)**: This feature allows users to export selected mesh objects from Blender to an IPL file in the format used by GTA San Andreas. It includes support for incremental model IDs and optional default rotations.
- **Export to IPL (GTA III)**: Users can export objects to an IPL format compatible with GTA III, accommodating the specific needs of this game's modding framework.
- **Export to IDE**: This feature enables exporting of object definitions to an IDE file, which is used to define object properties and settings within the game.

### Export to GTA VC IPL (In Development)
Currently, the addon includes basic functionality for exporting to the GTA Vice City (VC) IPL format. However, this feature is still under development and may not yet meet the final specifications required for GTA VC modding. The current implementation of the GTA VC IPL export function is designed to follow the general structure of the format, but additional adjustments and refinements are planned to fully align with the VC requirements.

**Note**: The GTA VC IPL export feature is in development and may not yet produce the exact output required for some VC modding scenarios. Future updates will aim to enhance this functionality and ensure full compatibility with the game's needs.

### Installation
1. **Download** the latest version of the addon.
2. **Open Blender** and go to `Edit` > `Preferences`.
3. Navigate to the `Add-ons` tab and click on `Install`.
4. Select the downloaded `.zip` file and click `Install Add-on`.
5. Enable the addon by checking the box next to `GTASceneSync IPL-IDE` in the list.

### Usage
1. **Access the Addon**: The addon adds a new menu under `File` > `Export`, where you'll find options to export selected objects in the various formats supported.
2. **Select Objects**: Choose the objects you want to export in Blender.
3. **Export**: Use the appropriate export option from the menu. For the GTA VC IPL format, ensure to review the output file as the feature is still in development.
